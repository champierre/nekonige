## 自分のマシンでスライドを開くには

```
git clone git@github.com:champierre/nekonige.git
npm install
npm run dev
```

を実行し、http://localhost:3030 をブラウザで開く。

slides.md がスライドの中身です。

Slidevについては[documentations](https://sli.dev/)を参考にしてください。
